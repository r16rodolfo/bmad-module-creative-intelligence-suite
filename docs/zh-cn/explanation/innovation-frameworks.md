Victor 的创新策略工作流运用经过验证的商业和创新框架，发现颠覆性机会并设计可持续的竞争优势。

## 什么是创新框架？

创新框架是发现和获取新价值的结构化方法。不同于头脑风暴（生成想法）或设计思维（解决用户问题），创新框架聚焦于商业模式创新和战略定位。

| 框架 | 核心问题 |
| ---- | -------- |
| **Jobs-to-be-Done** | 客户实际想要完成什么？ |
| **蓝海战略** | 未被争夺的市场空间在哪里？ |
| **颠覆性创新** | 我们如何从边缘发起进攻？ |
| **商业模式画布** | 我们如何创造和获取价值？ |

## Jobs-to-be-Done

**核心洞察：** 客户"雇佣"产品来完成一项工作。理解这项工作能揭示产品功能忽略的创新机会。

| 传统思维 | Jobs-to-be-Done 思维 |
| -------- | -------------------- |
| "千禧一代想要健康零食" | "人们需要在两餐之间保持高效" |
| "增加更多蛋白棒" | "让持续获取能量变得简单" |

### 框架

1. **识别工作** — 客户想要完成什么？
2. **理解上下文** — 何时、何地、有什么约束？
3. **发现痛点** — 当前方案在哪里失败？
4. **围绕工作创新** — 如何更好地完成这项工作？

:::tip[Victor 的方法]
Victor 不断追问："他们雇佣这个产品来做什么工作？"答案往往揭示了真正的机会空间。
:::

## 蓝海战略

**核心洞察：** 不要在拥挤的"红海"中竞争——创造"蓝海"般的未被争夺的市场空间。

| 红海 | 蓝海 |
| ---- | ---- |
| 在现有市场竞争 | 创造新市场 |
| 击败竞争对手 | 让竞争变得无关紧要 |
| 利用现有需求 | 创造新需求 |
| 对齐成本/价值权衡 | 打破权衡 |

### 框架

1. **消除** — 哪些因素应该被移除？
2. **减少** — 哪些应该低于行业标准？
3. **提升** — 哪些应该高于行业标准？
4. **创造** — 哪些从未存在过的应该被创造？

**示例：** 太阳马戏团消除了动物和明星表演者，减少了节目种类，提升了艺术和音乐质量，创造了全新的成人剧场体验。

## 颠覆性创新

**核心洞察：** 在位者过度服务主流客户，忽视了部分细分市场。颠覆者进入被忽视的细分市场并不断改进。

| 颠覆类型 | 模式 |
| -------- | ---- |
| **低端颠覆** | 服务在位者忽视的客户，然后向上发展 |
| **新市场颠覆** | 用更简单、更便捷的方案服务非消费者 |

### 框架

1. **识别被过度服务的客户** — 谁得到的比需要的多？
2. **找到非消费者** — 谁无法使用当前方案？
3. **简化** — 什么可以被移除同时仍然交付价值？
4. **快速改进** — 随时间向上游发展

:::caution[不是所有创新都是颠覆]
真正的颠覆遵循特定模式。大多数"颠覆性"的说法只是增量改进。Victor 区分真实与话术。
:::

## 商业模式画布

**核心洞察：** 你如何创造和交付价值与你创造什么同等重要。商业模式创新胜过产品创新。

### 九大模块

| 模块 | 问题 |
| ---- | ---- |
| **价值主张** | 我们交付什么价值？ |
| **客户细分** | 我们服务谁？ |
| **渠道** | 我们如何触达他们？ |
| **客户关系** | 我们如何互动？ |
| **收入来源** | 我们如何赚钱？ |
| **核心资源** | 我们需要什么资产？ |
| **关键活动** | 我们必须做什么？ |
| **关键合作** | 谁帮助我们？ |
| **成本结构** | 需要多少成本？ |

### 商业模式创新模式

| 模式 | 示例 |
| ---- | ---- |
| **免费增值** — 基础免费，高级付费 | Spotify、Dropbox |
| **订阅制** — 周期性收入 | Netflix、SaaS |
| **平台** — 连接买卖双方 | Uber、Airbnb |
| **刀片模式** — 便宜的硬件，昂贵的耗材 | 打印机、Keurig |

## 什么时候使用哪个框架

| 场景 | 最佳框架 |
| ---- | -------- |
| **理解客户动机** | Jobs-to-be-Done |
| **寻找未被争夺的空间** | 蓝海战略 |
| **分析竞争动态** | 颠覆性创新 |
| **设计/重塑商业模式** | 商业模式画布 |

Victor 经常组合使用框架。Jobs-to-be-Done 揭示机会；蓝海战略定位它；商业模式画布将其货币化。

## 创新 vs. 策略

| 创新 | 策略 |
| ---- | ---- |
| 创造新价值 | 获取已创造的价值 |
| 做什么 | 如何赢 |
| 选项和替代方案 | 选择和承诺 |
| 发散思维 | 收敛选择 |

CIS 的创新策略工作流衔接两者——先生成选项，再做出战略选择。
