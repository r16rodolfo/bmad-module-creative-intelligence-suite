Carson 的头脑风暴工作流提供 7 个类别 36 种技术，源自数十年的引导实践与研究。

## 为什么技术选择很重要

不同技术服务于不同目的：

| 目标 | 最佳技术类别 |
| ---- | ------------ |
| **快速生成大量想法** | 协作类、疯狂类 |
| **突破常规思维** | 创意类、戏剧类 |
| **深入理解问题** | 深度类、内省类 |
| **系统性探索选项** | 结构类 |
| **建立团队共识** | 协作类 |

## 技术类别

### 协作类技术

借助群体智慧和集体创造力。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **Brainwriting** | 静默书写构思，然后在他人想法上叠加 | 有强势发言者的团队 |
| **Round Robin** | 每人轮流贡献一个想法 | 需要平等参与 |
| **Six Thinking Hats** | 从六个角度探索（事实、情感、谨慎等） | 复杂、多面向的问题 |
| **Brainwalking** | 写想法、轮换站点、在发现的基础上构建 | 高能量的团队会议 |

### 结构类技术

不遗漏任何创意石头的系统性方法。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **SCAMPER** | 替代、结合、适应、修改、另作他用、消除、逆转 | 改进现有产品/服务 |
| **逆向头脑风暴** | 如何导致问题，然后翻转为解决方案 | 被常规方法困住 |
| **形态分析** | 参数矩阵，系统性组合 | 多变量的复杂系统 |
| **分块法** | 分解成小部分，分别构思，再重新组合 | 大型、令人不知所措的挑战 |

### 创意类技术

激发意外联想的非常规方法。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **隐喻** | "这个问题像一个花园吗？" | 需要新视角 |
| **随机词** | 强制在主题和随机词之间建立联系 | 完全卡住，需要惊喜 |
| **强制联想** | 组合两个不相关的概念 | 通过并置实现创新 |
| **类比思维** | 从其他领域借用解决方案 | 跨行业启发 |

### 深度类技术

深入表面理解之下，发现根因和隐藏的机会。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **五个为什么** | 连续问五次"为什么"来深挖 | 快速根因探索 |
| **阶梯法** | 上移到抽象，下移到具体 | 理解价值和细节 |
| **假设浮现** | 列出并挑战隐藏的假设 | 常规思维阻碍进展 |
| **问题逆转** | 反向陈述问题，翻转为解决方案 | 用新眼光看问题 |

### 戏剧类技术

通过角色扮演和场景思维绕过心理障碍。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **角色风暴** | 以他人身份构思（CEO、孩子、竞争对手） | 需要不同视角 |
| **英雄之旅** | 将挑战框架化为叙事冒险 | 有意义的、以人为本的挑战 |
| **未来新闻稿** | 像未来的成功已经发生一样写作 | 愿景和目标设定 |
| **预验尸** | 假设失败，逆向追溯原因 | 风险识别和预防 |

### 疯狂类技术

拥抱荒谬来绕过内在审查，找到真正新颖的想法。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **最糟糕的想法** | 故意生成糟糕的想法 | 害怕出坏主意的团队 |
| **反问题** | 解决你问题的反面 | 重新定义顽固的挑战 |
| **夸张法** | 把某些方面放大到荒谬的程度 | 找到真正重要的东西 |
| **100 个想法** | 在 20 分钟内生成 100 个想法 | 突破到新颖领域 |

### 内省类技术

通过个人反思调动潜意识创造力。

| 技术 | 工作方式 | 适用场景 |
| ---- | -------- | -------- |
| **引导想象** | 对挑战进行视觉冥想 | 获取直觉知识 |
| **孵化** | 将问题放一边，让潜意识工作 | 有意识的努力已经耗尽 |
| **梦境捕捉** | 记录并探索梦境中的洞察 | 需要潜意识素材 |
| **意识流** | 不停下来也不编辑地写 | 绕过内在批评者 |

## Carson 如何推荐技术

Carson 在建议方法时考虑三个因素：

| 因素 | 问题 |
| ---- | ---- |
| **你的目标** | 什么结果最重要？ |
| **你的上下文** | 团队还是个人？有多少时间？ |
| **你的能量** | 精力充沛还是疲惫困顿？ |

基于你的回答，Carson 推荐最可能产生突破的技术。

## 组合技术

要进行全面探索，可以按顺序尝试多种技术：

1. **先结构化** — 用 SCAMPER 或逆向头脑风暴来绘制空间
2. **再创意化** — 用隐喻或随机词来制造惊喜
3. **最后疯狂** — 用最糟糕的想法来打破剩余的约束

每种技术通往不同的创意路径。按序使用能创造更丰富的成果。
