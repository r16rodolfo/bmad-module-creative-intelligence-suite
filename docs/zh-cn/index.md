# Creative Intelligence Suite

一套用于创新、头脑风暴、设计思维和创意问题解决的智能体与工作流集合。

---

## 什么是 CIS？

Creative Intelligence Suite（CIS）为 BMad Method 扩展了"模糊早期"阶段的工具——在这个阶段，创意萌生、问题被重新定义、解决方案通过结构化创造力涌现。

## 可用智能体

| 智能体/工作流 | 用途 |
|---------------|------|
| **Innovation Strategist** | 发现颠覆性机会和商业模式创新 |
| **Design Thinking Coach** | 通过共情、构思和原型设计实现以人为本的设计 |
| **Brainstorming Coach** | 使用经过验证的技术引导创意构思会议 |
| **Problem Solver** | 系统性问题诊断与根因分析 |
| **Creative Problem Solver** | 运用横向思维生成创意解决方案 |
| **Storyteller** | 为产品和功能打造引人入胜的叙事 |
| **Presentation Master** | 构建并呈现有说服力的演示 |
