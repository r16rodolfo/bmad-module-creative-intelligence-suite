使用 `brainstorming` 工作流，通过 7 个类别 36 种经过验证的技术生成多样化、有创意的想法。

## 什么时候使用

- 启动新项目，需要多个选项
- 被某个问题困住，想要新视角
- 在确定方向之前探索各种方案
- 为未来考虑建立创意储备
- 引导团队构思会议

## 什么时候跳过

- 你已经有清晰、唯一的解决路径
- 时间极度紧张（不到 10 分钟）
- 问题需要分析性而非创造性的方法

:::note[前置条件]
- **已安装带 CIS 的 BMad Method** — Carson 是 Creative Intelligence Suite 的一部分
- **一个主题或挑战** — 你想头脑风暴的东西
- **对疯狂想法的开放态度** — Carson 鼓励非常规思维
:::

## 步骤

### 1. 加载 Carson

开始新对话并加载 Brainstorming Coach：

```
/cis-brainstorm
```

### 2. 定义你的主题

Carson 会问你想头脑风暴什么。提供一个清晰但开放的主题：

**好的主题：**
- "改善用户引导流程的方法"
- "预算应用的新功能"
- "内容平台的收入来源"

**效果较差：**
- "修复第 42 行的 bug"（太具体了）
- "应该用 React 还是 Vue？"（这是决策，不是构思）

### 3. 选择你的方式

Carson 提供四种选择头脑风暴技术的方式：

| 模式 | 适合场景 | 说明 |
| ---- | -------- | ---- |
| **用户选择** | 你知道想用哪种技术 | 浏览 36 种技术并挑选 |
| **AI 推荐** | 你不确定哪种最合适 | Carson 分析你的主题并推荐 |
| **随机** | 偶然发现和惊喜 | 让运气来选择 |
| **渐进式** | 全面探索 | 按顺序尝试多种技术 |

### 4. 用技术探索

Carson 使用"Yes, and..."方法论引导你：

- **在想法上叠加** — 不要评判，在已有基础上添加
- **追求数量** — 更多想法提高质量概率
- **拥抱疯狂** — 疯狂的想法往往藏着珍宝

:::tip[Carson 的风格]
Carson 像一个充满热情的即兴表演教练——高能量，为每一个贡献喝彩，保持动力持续流动。
:::

### 5. 捕获和精炼

构思结束后，Carson 帮你：

1. **聚类相似想法** — 发现主题和模式
2. **标记亮点** — 标出值得追求的想法
3. **添加实践细节** — 让可执行的想法更具体

## 你将获得

输出保存到 `_bmad-output/brainstorming-{date}.md`：

| 部分 | 内容 |
| ---- | ---- |
| **主题** | 你的头脑风暴挑战 |
| **使用的技术** | 应用了哪种方法 |
| **生成的创意** | 所有想法的完整列表 |
| **精选推荐** | Carson 的推荐 |
| **下一步** | 如何推进 |

## 示例

```text
你：/cis-brainstorm
Carson：你想头脑风暴什么？
你：减少购物车放弃率的方法
Carson：太棒了！让我们用逆向头脑风暴。
       不问"如何减少放弃率"，而是问：
       "我们如何最大化放弃率？"
你：[生成逆向想法]
Carson：很好！现在让我们翻转这些：
       - "让结账流程令人困惑" → "简化到 3 次点击"
       - "隐藏运费" → "预先显示总价"
       [从逆向角度生成可执行的解决方案]
```

## 技巧

:::tip[最佳实践]
- **准备上下文** — 如果主题复杂，准备好背景信息
- **设置计时器** — 每种技术 15-30 分钟，保持高能量
- **记录一切** — 即使奇怪的想法后来也可能激发灵感
:::

:::caution[避免常见错误]
- 在生成阶段不要批评想法——那是之后的事
- 在完整列表出来之前不要急着挑最喜欢的
- 不要用头脑风暴来做需要数据而非创意的决策
:::

## 可用技术

Carson 的 36 种技术横跨 7 个类别：

| 类别 | 包含的技术 |
| ---- | ---------- |
| **协作类** | Brainwriting、Round Robin、Six Thinking Hats |
| **结构类** | SCAMPER、逆向头脑风暴、形态分析 |
| **创意类** | 隐喻、随机词、强制联想 |
| **深度类** | 五个为什么、阶梯法、假设浮现 |
| **戏剧类** | 角色风暴、英雄之旅、未来新闻稿 |
| **疯狂类** | 最糟糕的想法、反问题、夸张法 |
| **内省类** | 引导想象、孵化、梦境捕捉 |
