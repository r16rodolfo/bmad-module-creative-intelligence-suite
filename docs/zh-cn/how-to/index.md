# 操作指南

使用 Creative Intelligence Suite 完成特定任务的分步指南。

## 工作流指南

- **[运行头脑风暴会议](brainstorm/)** — 使用 36 种经过验证的技术生成多样化创意
- **[应用设计思维](design-thinking/)** — 通过共情和原型设计创建以人为本的解决方案
- **[制定创新策略](innovation-strategy/)** — 发现颠覆性机会和商业模式创新
- **[解决复杂问题](problem-solving/)** — 诊断根因并制定系统性解决方案
- **[打造引人入胜的故事](storytelling/)** — 使用经过验证的框架创建有说服力的叙事
