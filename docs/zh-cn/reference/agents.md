CIS 智能体的完整目录，包括命令、用途和核心能力。

## 智能体目录

### Carson — 头脑风暴教练

**用途：** 使用 7 个类别 36 种经过验证的技术生成多样化创意。

| 属性 | 值 |
| ---- | -- |
| **图标** | 🧠 |
| **工作流** | `brainstorming` |
| **命令** | `/cis-brainstorm` |
| **专长** | 群体动力学、心理安全感、鼓励疯狂想法 |

**菜单触发器：**
- `BS` 或模糊匹配 `brainstorm`

**核心能力：**
- 7 个类别 36 种构思技术
- 基于主题和上下文的技术推荐
- 使用"Yes, and..."方法论在想法上叠加
- 渐进模式用于全面探索

### Sophia — 叙事师

**用途：** 使用经过验证的叙事框架打造引人入胜的故事。

| 属性 | 值 |
| ---- | -- |
| **图标** | 📖 |
| **工作流** | `storytelling` |
| **命令** | `/cis-storytelling` |
| **专长** | 25 种叙事框架、情感心理学、叙事弧线 |
| **Sidecar** | 是 — 记住故事偏好和历史 |

**菜单触发器：**
- `ST` 或模糊匹配 `story`

**核心能力：**
- 25 种叙事框架（英雄之旅、Story Brand、三幕式等）
- 情感节拍打造
- 平台特定适配
- Sidecar 记忆用于个性化

**关键动作：**
- 从 sidecar 加载用户偏好
- 审查故事历史以保持一致性
